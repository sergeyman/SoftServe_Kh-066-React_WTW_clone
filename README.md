# WHAT TO WATCH (WTW)

## About
WTW this is a web application that displays a list of movies built using the React JS library.

## For review or development locally

1. Clone repository
    `git clone https://github.com/kerrimov/what-to-watch`

2. Install dependencies
    `yarn install`

3. Run
    `yarn start`

4. Test
    `yarn test`

5. Build
    `yarn build`
